# Codex Settings Export

This directory exports the current local Codex configuration from `~/.codex`.

Included:
- `config.toml`
- `AGENTS.md`
- `agents/*.toml`
- `rules/default.rules`

Excluded for security/privacy:
- `auth.json`
- `history.jsonl`
- `sessions/`
- `state_*.sqlite*`
- `log/`
